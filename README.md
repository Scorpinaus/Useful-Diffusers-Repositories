# Useful-Diffusers-Repositories
This is a compiled list of useful github repositories which enhances the functionality of Huggingface Diffusers

sd_embed - Helper repo to generate long weighted prompt embeddings for SD1.5, SDXL, SD3 and Flux pipelines
https://github.com/xhinker/sd_embed

attention-map-diffusers - Repository to extract and visualise cross-attention maps
https://github.com/wooyeolbaek/attention-map-diffusers

Hugging Face Inference Toolkit - Official Huggingface Repository for containerzation
https://github.com/huggingface/huggingface-inference-toolkit

HunyuanDIT-Img2Img - 
https://github.com/maepopi/HunyuanDIT-Image-to-Image

RF-Image-Editting - 
https://github.com/Justin900429/RF-Image-Editing

stable-fast -
https://github.com/chengzeyi/stable-fast

controlnet_aux - Pypi installable package of Illyaviel ControlNet Annotators (Not maintained)
https://github.com/huggingface/controlnet_aux

compel - Prompting enhancement repo 
https://github.com/damian0815/compel
